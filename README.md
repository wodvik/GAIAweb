# GAIAweb

Visualisation and educational website for the probabilistic Gaia DR3 catalogue of stars with very low Galactic rest-frame speeds (slow \(V_{\mathrm{grf}}\)).

## Related work

- **Catalogue repository:** [wodvik/gaia-slow-vgrf-catalogue](https://github.com/wodvik/gaia-slow-vgrf-catalogue)
- **Zenodo release:** [doi:10.5281/zenodo.21230323](https://zenodo.org/records/21230323)
- **Concept DOI:** [10.5281/zenodo.20116134](https://doi.org/10.5281/zenodo.20116134)

## About the catalogue

A Probabilistic Gaia DR3 Catalogue of Stars with Very Low Galactic Rest-Frame Speeds (Humble, 2026). Headline Tier A+B sample: stars with high probability of \(V_{\mathrm{grf}} < 25\,\mathrm{km\,s^{-1}}\).
